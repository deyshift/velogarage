# VeloGarage — Claude working notes

## Pull requests
- **Always include a GitHub closing keyword in the PR body** so the linked
  issue auto-closes when the PR merges. Use `Closes #N` / `Fixes #N` /
  `Resolves #N` on its own line. A bare reference like `(#N)` in the title or
  commit message does **not** close the issue — it only links it.
- When the work resolves an existing issue, name that issue with a closing
  keyword in the PR description, not just the branch/commit.

## AI attribution
- **Never put Claude session links** (`claude.ai/code/session_…`) in commit
  messages, PR titles, or PR descriptions.
- Attribute AI-generated work with a single simple line — `🤖 AI-generated by
  Claude` — at the end of commit messages and PR bodies. Nothing more: no
  session URL, no model version, no `Co-Authored-By` trailer.
